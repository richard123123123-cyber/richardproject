# 網站網址  https://richardproject-nine.vercel.app/

**關於網站**：
這個網站裡面主要是關於舊衣服買賣網站 裡面可以看到的是關於製作網站的練習
網站裡面有用關於ＣＳＳ ＨＴＭＬ
有用到不同的排版技巧 像是文字擺放位置和圖擺放位置

**網站內部**：
在`practice`裡面還有個`rainbow.html`的是關於排版和文字大小顏色練習
在`practice`裡面有`design`裡面有設計圖
在`index.html`裡面有網站的程式碼

**網站介紹**：
在網站中 上半部左邊有網站的logo 右邊是加入,創建帳號和幫助
網站中部分是關於網站的分類 可以選擇男人 女人 還有小孩選項 還有外套 短褲短袖 長褲長袖 鞋子 帽子的分類
網站的底下部分是六個最新上架的商品 有商品名稱 商品的價格 還有屬於的分類
